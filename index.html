Shoot'em up Javascrip

Semestre 4

Ce projet est un shoot'em up écrit en JavaScript.
Dans ce jeu, nous contrôlons un personnage, et nous devons survivre aux ennemies qui apparaissent en boucle sur le jeu.
Nous pouvons pour cela tirer des projectiles pour nous défendre, et récupérer des bonus pour survivre plus longtemps.
Ce jeu est multijoueur, plusieurs personnes peuvent se connecter en même temps grâce à un serveur qui s'occupe de tous les calculs.

Outils : Javascript, HTML, CSS, méthode agile
Savoir-être acquis : Organisation

API Rest

Semestre 4

Lors de cette SAE, nous devions mettre en place une API REST pour une pizzeria.
Ainsi, nous devions des endpoint, c'est-à-dire des url qui permette de récupérer des informations ou les modifié depuis une console ou un navigateur.
Nous avons aussi mis en place une sécurité avec un token d'accès.

Outils : Java, SQL, Tomcat, HTML
Savoir-être acquis : Organisation

Chasse aux monstres

Semestre 3

Le jeu Chasse au Monstre est un jeu programmé en java où 2 joueurs s'affrontent dans un labyrinthe
Un monstre, qui a pour but de s'échapper du labyrinthe qu'il connaît entièrement,
et un chasseur qui doit trouver le monstre dans un labyrinthe qu'il ne connaît pas.
Les joueurs jouent chacun à tour de rôle.

Outils : Java, méthode agile
Savoir-être acquis : Travail d'équipe

Application d’échange internationale

Semestre 2

L'application d'échange était un projet de création d'application.
Nous devions créer une application permettant de mettre en lien des élèves de différents pays dans le cadre d'un échange international.
Il y avait un côté élève, où l'utilisateur devait remplir un formulaire pour s'inscrire, et un côté professeur pour mettre en relation les élèves ayant le plus d'affinités.

Outils : Java, Javafx, fxml, SQL
Savoir-être acquis : Communication
